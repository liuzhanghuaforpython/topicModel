说到纯电动汽车就不得不提特斯拉了，作为纯电动汽车的先行者，特斯拉可以说是家喻户晓。但是一路走来，特斯拉同样经历了很多跌宕起伏，比如高管离职、Model 3 产能不足、使用自动辅助驾驶系统导致车祸等问题。而其中 Model 3 的产能一直是被重点讨论的对象。今天，新出行就先和大家一起了解下特斯拉 Model 3 产能的问题吧。
1、安抚民心 特斯拉 Model 3 首批交付 30 台
在经历了几次跳票之后，北京时间 2017 年 7 月 29 日，特斯拉在美国加州举办了首批特斯拉 Model 3 交车仪式，此次交车的对象为特斯拉内部员工和北美的 VIP，共计交付 30 台。
此次交付的对象是内部员工及北美 VIP 客户，对于此次交付的 30 台 Model 3 来说，此举更像是为了证明自己可以量产出 Model 3 车型，给那些观望中的消费者打一针“镇定剂”。
2、产能持续低迷 第三季度交付 220 辆 Model 3
在第三季度财报披露前几天，伊隆马斯克表示 9 月底Model 3 产能可以达到 375 辆/周，虽然 375 辆/周的产能不是很理想，但还是有进步的。结果从公布的第三季度汽车交付情况来看，特斯拉仅生产了 260 辆 Model 3，交付了 220 辆。如此低的 Model 3 交付量立刻造成了特斯拉的股价下跌。而后，马斯克在推特上宣布为了集中力量生产 Model 3 旗下的卡车发布会也将推迟举行。
那么，第三季度 Model 3 产能为何会有如此大的产能落差？后来据松下CEO透露，Model 3产能不达标与电池有关。由于电池自动化生产线投产时间推迟，导致电池生产的部分环节需要靠手工来完成，从而导致了 Model 3 的产能没达预期。
3、第四季度产能提升 周产千台 Model 3 产能回暖
在第四季度财报发布之前，特斯拉成功的将一台红色的 Roadster 送上了太空，此举依然无法转移 Model 3 产能带来的不利消息。从公布的 2017 年度第四季度的汽车交付情况来看，特斯拉在第四季度共生产 2425 台 Model 3,交付量仅为 1550 辆，解决了所谓的生产瓶颈后， Model 3 产量依然不够理想。
据媒体透露，影响第四季度 Model 3 产能的主要原因依然是电池组装，在电池自动化生产线上设有 4 个环节，最初的 1-2 环节是外包给了供应商，由于供应商设计的产线在大规模量产后无法满足特斯拉的要求。最后，特斯拉团队决定自己在 6 到 9月重新研发新的自动化生产线。
4、产能爬坡 2018 年第一季度周产达 2000 台
到了 2018 年 4 月，特斯拉发布了 2018 年第一季度产销数量。其数据显示，2018 年第一季度 Model 3 交付了 8180 辆，是上一季度的 4 倍。而且 Model 3 产能已经连续三周超过 2000 辆/周，并且在 4 月达到了 2270 辆/周，但依然没有达到马斯克之前承诺的前 3 月低实现 2500 辆/周的产能。
第一次产能不达标是因为电池组装问题，第二次不达标依然是电池占主要责任，但 Model 3 的产能一直无法达标真的全部因为电池吗？有消息称，Model 3 产能无法达标和总装自动化有关系，由于总装自动化率上不去，特斯拉拟增加人手来实现 7*24 小时的三班倒模式。
特斯拉 Model 3 产能总结：
2017 年 7 月首批交付 30 台
2017年第三季度生产 260 台，交付 220 台
2017年第四季度生产 2425 台，交付 1550 台
2018年第一季度交付8180台
截止 2018年第一季度末，特斯拉共生产 10895 台 Model 3
以上为特斯拉 Model 3 的产能及交付信息，整体看来 Model 3 的产能一直属于严重不达标状态，截止 2018年第一季度末，特斯拉共生产 10895 台 Model 3。很快 2018 年第二季度财报就会出来，到时候就知道 Model 3 的最新产能信息了。
5、产能提速 中国 Model 3 车主或明年提车
在5 月 21 日，马斯克公布了双电机版、Performance 版 Model 3 的售价和参数。同时，马斯克表示双电机版本的 Model 3 并不会在原有的装配线上组装，而是在加州弗里蒙特的特斯拉工厂为 Model 3 铺设了一条新的装配线。
5 月 21 日之前的订车信息显示需要等待 12-18 个月才能交付
5 月 21 日后的订车信息显示只需等待 4-6 个月就能交付
随后，特斯拉官网就更新了 Model 3 车型介绍。而且官网的预定后提车时间也有所变动，由当初的订车后等待 12-18 个月变成了 4-6 月就能交付，双电机版本的需要等待6-9个月，这些信息显示出 Model 3 产能在回归正轨。
翻译：Model 3 为对称性设计，所以生产右舵车型只需很少的改动就可以，2019 年上半年交付亚洲和欧洲后，年中就可以交付右舵版本。
通过官网的订车信息及马斯克在推特上表示来看，今年年底将完成北美的 Model 3 车辆交付。北美交付完成后将在2019年上半年开始亚洲和欧洲的 Model 3 车辆交付，做为特斯拉最重要的市场之一，中国应该是亚洲首批交付的国家。所以国内 Model 3 准车主最快将在 2019 年上半年拿到自己的爱车。
最新邮件部分内容翻译：Model 3 生产系统所有环节均超过了 500 辆/天，部分可达 700 辆/天，也就是 5000 辆/周。虽然达到了 5000 辆/周，但是特斯拉 Model 3 依然存在产能瓶颈的问题，其中涂装工艺、焊装工艺、电池组模组等还需要继续改进和提升。
新出行点评：
通过几次季度报来看，特斯拉 Model 3 截止第一季度为止共生产 10895 台，整个 Model 3 的产量和交付量受电池和总装的影响一直未能达标。但从近期双电机版本 Model 3 的亮相及官网的订车信息的来看，特斯拉 Model 3 的产能正在逐步回归正轨。因此年底完成北美的交付后就可以开始亚洲欧洲的交付，所以国内的 Model 3 车主应该不需要等太久。
