原标题：EU5五智十谈 | 八大优点铸就的好电池
 
对于纯电动汽车来说，动力电池自然是其最核心的部件，也是影响车辆续航的关键。作为北汽新能源新近推出的人工智能轿车新典范，EU5就由于搭载了高性能三元(报价 参数 图片 论坛)锂电池，从而为车辆带来了持久续航，最高续航达到570公里，堪称目前国内纯电动汽车产品的续航翘楚。其中R500车型，工况续航416公里，等速续航520公里；R550车型，工况续航460公里，等速续航570公里。

据了解，EU5所采用的高性能三元锂电池，选取了CATL宁德时代全新一代电芯，并通过北汽新能源9年来累积的行业领先电池打包封装技术，将电芯、模块、电气系统、热管理系统、壳体和电池管理系统几个部分完美组装，最终形成电池系统整体，并在多重防护配置的调控下长期、高效、稳定运转。今天，小编就带你走近EU5的的动力电池，揭开其拥有的八大优点。

n 优点一：能量密度大
由于采用了宁德时代的全新一代电芯技术，EU5的高性能三元锂电池在保证寿命长的同时，还大幅提升了电池能量密度，高达151Wh/kg，在行业中处于领先水平。由此也享受到了1.1倍的国家补贴，降低了车价，让消费者享受了更多优惠。
 
n 优点二：电量高
电池能量密度大，同样的电池体积则可装下更多的电量。EU5 的R500车型电池容量达到了53.6kWh，R550车型电池容量达到了60.2 kWh。俗话说，一分电量一分续航，更大的电池容量为EU5拥有领先同级的超长续航奠定了基础。
 
n 优点三：安全可靠
装车之前，EU5的动力电池就已历经30项国家标准验证和52项企业标准验证，其中包括：过充、过放、过温、短路、跌落、挤压、火烧、机械冲击、模拟碰撞、盐雾等严苛检验项目。而装车之后，EU5还通过了高温、高寒、高原等各种极端情况超百万公里的模拟测试，全面验证了电池及整车的可靠性，从而保证车型量产之后的用户使用安全。

 
n 优点四：适应性强
三元锂电池在不同工作环境下活性会有所改变，这也造成了不同环境下充电效率的差异性。而在EU5的EMD 3.0技术中，集成了多项电池活性辅助功能，极大地优化了车主在日常充电过程中的便利性。例如：
低温预加热功能：当在低温环境下为EU5充电时，电池会在加热包的帮助下，进行预加热，使电池尽快达到正常工作温度，提升电池充电效率，为用户节省充电时间。
边充电边加热：当动力电池温度过低时，车辆可在充电过程中，对动力电池持续加热，使其在低温环境下充电效率更高。
慢充保温：当动力电池温度较低时，插枪状态下可对动力电池进行保温，避免充电结束后由低温导致的电池可用容量损失。
整车双向充电：EU5高配车型的动力电池可以作为一个大的移动电源，为负载用电器供电，或者为其他车辆充电，为用户提供更多样的使用方式。
动力电池液冷：EU5高配车型可监控电池电芯温度以及冷却液温度，合理地控制电池系统处于加热状态还是冷却状态，确保动力电池在各种季节温度下，都能保持高效运转。
与此同时，EU5电池系统的防护等级达IP67潜艇级安全标准，防尘防水，稳定可靠。

 
n 优点五：智能度高
由于装载了EMD3.0超级电控技术，EU5电池控制系统能使其电池具有多重智能安全保护措施，如电池实时预测和报警、快充快放保护、过充过放安全防护、低温充电预加热等功能，不仅保护了电池的安全，更保障了消费者的人生安全。
 
n 优点六：寿命长
EMD3.0超级电控技术，还使EU5具备了电池单体电压偏差自均衡功能，这个功能带来的最大好处就是会自动甄别判断电池组中电池单体的电压偏差，并根据异常情况，自动进行均衡处理，使电池单体电压偏差保持在预期的范围内，帮助车辆保持超长续航，并有效延长电池使用寿命。正因此，北汽新能源敢为用户提供8年或15万公里后，电池容量依然高达80%的电池质保承诺，让用户使用更安心。

 
n 优点七：续航持久
目前，在售的纯电动汽车中，等速续航突破500公里的车型寥寥无几。EU5 R500车型工况续航416公里，等速续航520公里；R550车型工况续航460公里，等速续航570公里。这样的续航表现傲视同级，可为用户打造出500+公里半径电动生活圈，让更多人都能享受到绿色出行的优势。
上下班通勤是最常见的用车场景，以国内交通最繁忙的城市之一――北京为例，北京市政府2015年公布的统计结果显示，市民工作日车辆日均往返行驶里程约41.5公里。而EU5即使在复杂路况下的续航能力也可达416公里，足可以满足用户连续使用近10天，从而确保大多数人使用一周只需充一次电。

 
n 优点八：充电便捷
在EMD3.0的电池控制系统调配下，EU5可提供三种充电方式：其中快充可在公用快充桩上，实现30分钟内电量从30%充至80% ; 慢充则给所有有安桩条件的消费者，免费赠送和安装充电桩，6-8小时充满，晚上充电白天用；而随时充的状态下，消费者还可自行选购3.3kW或1.8kW随车应急充电线，220v家用电即可充。多元化的充电模式选择，让用户随时随地充电，不仅方便，且更为安全。
 
伴随着国内充电环境的日趋成熟，当前国家电网已建成“九纵九横两环”高速城际快充网络，覆盖19省、150个城市，建设充电桩5.6万个，里程超过3.1万公里。在这样的高速充电网络下，EU5已不再局限于续航能力范围内的出行区域，与传统燃油车相比，在使用便利性上已无差异。而由于搭载了高性能三元锂电池，EU5也拥有了性能优良、续航持久、使用安全、出行便捷等特性，从而能够为用户带来更智慧，更便利的绿色出行体验。
