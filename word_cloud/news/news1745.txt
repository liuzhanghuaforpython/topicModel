刚刚看完今晚的“双牙之战”，虽然打成了3:3平手，但球迷们却直呼过瘾，明明只是一场小组赛，但两队再次交战的精彩甚至看出了四强赛的刺激感觉。按照这状态来看，作为葡萄牙老将的C罗在首场就成功完成“帽子戏法”，本届世界杯的最佳射手似乎也相当大概率了。
C 罗再次在世界杯成为球迷们关注的焦点，但在作为汽车媒体，笔者更关心的是赛前爆出来的C 罗“偷税漏税”时间处理的如何了？
据西班牙《世界报》的消息，C 罗与西班牙检方已经就自己的税务问题达成一致，他将承认自己存在的多达 1470 万欧元的税务问题，并将缴纳 1880 万欧元的罚款，接受两年的有期徒刑，但是不用坐牢。
C 罗偷税漏税跟汽车新闻有什么关系？
今年5月11日，WEY品牌官方宣布：国际足球巨星克里斯蒂亚诺・罗纳尔多（C罗）正式出任WEY品牌代言人，这也是长城首次大手笔用在广告营销费用上。
C 罗虽然躲过了牢狱之灾，但是WEY是否将受到更大的“牵连”？
C 罗这个人，人长的还不错，最主要的是球也踢的漂亮，私生活也是经常上娱乐新闻头条，高流量＋实力派，用“世界超级巨星”来形容 C 罗不算过分。所以谁能签下C 罗，C 罗就能送谁上头条。甚至皇马传奇萨尔加多认为C 罗是皇马的无价之宝,皇马将不会卖掉C罗。
在中国SUV界称王称霸的哈弗给了魏建军也是这么想的，所以一狠心就签下了C 罗，用其来代言长城旗下的高端SUV品牌WEY，那么接下是不是就等着WEY大卖了？
可是没想到的是，半路杀出了程咬金，魏建军当初也不知道C 罗没有养成按时缴税的好习惯，不过一心只想踢好自己最后一届世界杯的C 罗的认错态度很诚恳，也愿意接受罚款，所以并没有激起过多的新闻浪花。
至于会不会对WEY品牌造成什么负面影响，我们暂时还无法定论，毕竟这一个月的头条都是世界杯的。
代言有风险，企业需谨慎
从长城方面来看，签下C 罗算是狠了下心的。笔者关注的长城公布2017年财报显示，虽然全年累计销量达到1,070,161辆，与上年的同期销量基本持平，但归属于上市公司股东的净利润仅为50.35亿元，同比减少52.28%。长城给出的说法是：当时关于营业利润的减少的原因，对现有产品进行促销，和加大品牌及新产品推广力度导致。
长城本身在汽车广告推广上就比较保守，2017年除了几次媒体试驾，几乎没有什么特别的事件能让大众可以去回忆的，这或许就是长城在广告上如此大方的原因了。
“获刑”事件对 C 罗形象肯定是有一定的影响，但是在这个娱乐至死的年代，大众对超级巨星的宽容度还是很高的，像C 罗这样的实力派，或许几个漂亮的射门就能挽救形象，但请代言对企业来说，还是相当冒险的。
这类品牌形象代言人出事，车企背锅的例子还不止长城WEY。近年来，最“经典”的莫过于“林丹出轨事件”，作为日产的形象代言人，“超级丹”在2016年广州车展前一天就闹出了“出轨丑闻”，最后搞得日产的不得不重新设计车展展台，重新搭建展板等，林丹的形象照片也从展位和展板上消失。林丹形象受损后，东风日产也随即“走红”网络，但是却是以一种非常规的方式。
在广告营销届存在着数种营销方式，突然爆火的文案案例很多，但是这属于概率事件，在广告人的眼里，能达到最佳宣传效果的另外一种方式就是请明星了。以请明星代言，就指望能把品牌逼格或销量提高的行为其实是有些不聪明的。
第一，俗话说，羊毛出在羊身上，代言费无论高低，都要列入财务费用，最后买单的都是消费者。第二，明星代言的效果有多大？最典型的就是：曾经的观致品牌请来喊麦的网红“天佑”，给观致带来了多少销量？这都需要企业去算这笔帐。
如果真的想要自己的产品卖得好，就应该踏踏实实的造好车，卖出了口碑自然就火。而且哈弗SUV发展的如日中天，长城是有这个优势的。虽然请了“超级球星”C 罗代言，笔者还是觉得长城这钱花的不值。
